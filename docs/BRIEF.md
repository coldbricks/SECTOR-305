# SECTOR 305 — Product brief

**Working title:** SECTOR 305  
**Position fantasy:** `DADE A07 · SECTOR 305 · WATCH OPEN`  
**Tag:** *Miami on the map. Console on the glass.*

## What this is

**SECTOR 305** is a **certification-grade 911 / PSAP dispatch simulator** set in a **fictional Miami-area** communications center. The player works the **A-console** (primary radio / unit control), not arcade cop fantasy.

**North star:** South Beach **swag** on the shell; **ruthless instrument** on the glass. Complex systems. Not shit. Not GTA-with-a-phone.

**Sibling DNA (aesthetic / product family):** ChloeVibes, ChloeVR, Finally — night void, glass panels, neon as *earned* accent, cockpit/console language, EFIS-style status color discipline. **Do not** put hot-pink club chrome on the live CAD by default.

## What “no bullshit” means

### Do

- Doctrine first (priorities, status codes, radio format, jurisdiction friction)
- Grade **process** (wrong priority, bad unit, missed update, status hygiene) not “story won”
- Incomplete information (wrong address, language barrier, stepped-on radio, late unit updates)
- Real clock: received → entry → dispatch → en route → on scene → clear
- Checkride / debrief as first-class (timeline scrub, fail flags)
- Sexy by **restraint** (dark glass, rationed cyan/green/amber/red)

### Don’t

- Arcade chase points / superhero dispatch
- Omniscient perfect map
- Fake official APCO/NENA “certified” claims without partners (say *standards-aligned / training sim*)
- Gore tourism or exploitative crisis mini-games
- Neon CAD cosplay as the default working UI
- “Whole Miami empty” — prefer one deep sector over a hollow county

## Aesthetic law (non-negotiable)

| Layer | Look |
|--------|------|
| Brand / launcher / menu | South Beach night — void black, one neon hairline, confident type |
| Live A-console | EFIS / pro-console — navy glass; cyan = set/data; green = engaged; amber = caution; red = bad only |
| Debrief | Clinical evaluation form, not confetti |

**Shell can flirt. Console does not perform.**

South Beach theme = sexy by **swag**, not by design-deck cosplay. No purple gradients “because brand.” Color is earned.

## Product spine (roles)

1. **Call-taker (C-side)** — location, nature, structured interrogation → create CFS  
2. **A-console (main)** — own the channel, assign units, track status, radio discipline  
3. **B / assist / supervisor** — later  

**Session types:** Academy block · Watch (generated/busy shift) · Checkride (pass/fail) · Debrief playback  

**Cert tracks (in-product fiction, not real licenses):** CT → A-POL → A-FR → dual → major incident  

## Phase 0 scope (build first)

**Vertical slice only. Must not be thin.**

- One Miami **sector** (e.g. coastal / Central corridor fiction)
- **Police A-console** only
- Working surfaces: **incident queue · unit status board · radio panel · incident form · timers · (useful, imperfect) map**
- Fixed **radio format** + **status codes** + **priority table** (document as law)
- **One busy watch** + **one checkride scenario**
- **Debrief**: timeline of actions + graded fails
- Shell: SECTOR 305 login / sector open with South Beach swag; glass stays clinical

**Exit criterion:** A real telecommunicator would nod at the glass even if they argue details.

## Systems that must exist in Phase 0 (minimum complex)

1. **CFS lifecycle** with times and required updates  
2. **Unit statuses** that break if player is sloppy  
3. **Priority coding** that can fail the checkride  
4. **Closest / appropriate unit** logic (simple but real)  
5. **Radio exchange model** (template + required readbacks — voice optional later)  
6. **At least one bad-info scenario** (wrong block, language delay, or incomplete description)  
7. **Rubric**: binary fails + soft marks; evaluation form at end  

## Explicit non-goals for first pass

- Full county multi-agency map  
- Fire/EMS apparatus depth (Phase 2)  
- Real-time voice STT as required path  
- Co-op multi-console  
- Hurricane / active assailant as day-one content (design hooks OK; not required for slice)  
- Claiming official public-safety certification  

## Content caution

- Trauma-aware: structured, respectful modules; no shock porn  
- Clear disclaimer: **not a substitute for agency training / not official**  
- All locations, units, and incidents are **fictional training content**

## Success test

After a serious Phase 0 pass you should be able to:

1. Open a watch  
2. Work one messy call on the glass  
3. Fail for a **real** reason (e.g. dispatched without verify / wrong priority / unit left wrong status)  
4. Read a debrief that feels like training eval, not a game over screen  
5. Feel **South Beach** at the door and **console** at the job  

If any of those fail, go back to doctrine + core loop — not more features.
